Công nhận, nếu bạn đang dùng combo Mac Mini và iPad thì Apple Notes đúng là "chân ái" vì khả năng sync (đồng bộ) gần như tức thì và cực kỳ nhẹ máy.

Vì bạn là dân dev, chắc chắn bạn sẽ thích làm chủ ứng dụng này hoàn toàn bằng bàn phím. Dưới đây là bộ bí kíp phím tắt và mẹo format text "nhanh như chớp" trên Apple Notes macOS:

### 1. Dàn ý và Cấu trúc (Hierarchy)

Thay vì bôi đen rồi click chuột chọn size chữ, bạn hãy dùng tổ hợp `Cmd + Shift + Số` (hoặc Chữ) để phân cấp nội dung:

* **Tiêu đề lớn nhất (Title):** `Shift + Cmd + T`
* **Tiêu đề chính (Heading):** `Shift + Cmd + H`
* **Tiêu đề phụ (Subheading):** `Shift + Cmd + J`
* **Văn bản nội dung (Body):** `Shift + Cmd + B` (dùng để reset format về chữ bình thường).

### 2. Danh sách và Task (Lists & Checklists)

* **Tạo Checklist (Việc cần làm):** `Shift + Cmd + L`.
* *Mẹo:* Khi đứng ở một dòng checklist, nhấn `Cmd + Shift + U` để đánh dấu tick hoàn thành (hoặc bỏ tick) mà không cần với tay cầm chuột.


* **Danh sách dấu chấm (Bullet list):** `Shift + Cmd + 7` (Hoặc đơn giản là gõ `*` rồi bấm Space).
* **Danh sách gạch ngang (Dashed list):** `Shift + Cmd + 8` (Hoặc gõ `-` rồi bấm Space).
* **Danh sách đánh số (Numbered list):** `Shift + Cmd + 9` (Hoặc gõ `1.` rồi bấm Space).
* **Thụt lề (Indent/Outdent):** Dùng `Cmd + ]` để lùi vào trong và `Cmd + [` để đẩy ra ngoài (rất tiện khi làm list lồng nhau).

### 3. Định dạng văn bản chi tiết

Ngoài các phím cơ bản như `Cmd + B/I/U` (Đậm/Nghiêng/Gạch chân), bạn có thể dùng thêm:

* **Gạch ngang chữ (Strikethrough):** `Shift + Cmd + X` (Rất hay dùng để gạch bỏ code lỗi hoặc ý tưởng cũ).
* **Trích dẫn (Blockquote):** `Cmd + '` (Dấu nháy đơn). Nó sẽ tạo một đường kẻ sọc bên trái đoạn văn bản, nhìn rất giống blockquote trong Markdown.
* **Chèn Link (Hyperlink):** `Cmd + K`. Bạn bôi đen chữ, bấm `Cmd + K` rồi dán URL vào là xong, note sẽ rất gọn gàng.

### 4. Mẹo nâng cao: "Vũ khí bí mật" của Apple Notes

* **Chèn Bảng (Table):** `Option + Cmd + T`. Bạn có thể dùng phím `Tab` để tự động nhảy sang ô tiếp theo hoặc tự động tạo thêm hàng mới khi đến ô cuối cùng.
* **Liên kết các Note với nhau (Wiki-style):** Nếu bạn đang dùng macOS bản mới (từ Sonoma trở lên), bạn chỉ cần gõ **`>>`** trong note. Nó sẽ hiện ra danh sách các note khác của bạn để bạn chọn và link trực tiếp vào. Cực kỳ tiện để làm hệ thống tài liệu cá nhân (Personal Knowledge Management).
